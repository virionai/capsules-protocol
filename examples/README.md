# Capsule Examples

These examples are small, generic protocol fixtures. They demonstrate how
a capsule can carry portable work products such as reports, tables,
graphs, and rendered HTML without depending on a specific application or
external service.

No warranty: these examples are illustrative only. They are not production
templates, compliance artifacts, legal advice, security advice, or
operational guidance.

## Examples

- `generic-report/` builds a signed capsule containing a static report
  work product.
- `generic-table-graph/` builds a signed capsule containing a
  self-contained HTML table and graph.
- `generic-react-render/` builds a signed capsule containing static HTML
  rendered from a small React component at build time.

Each example is self-contained:

```sh
cd examples/generic-report
npm test
```

Generated capsules are written under each example's `output/` directory.
Those outputs are intentionally ignored by git and regenerated by tests.
