# Quickstart

The runnable version of the quickstart in
[`sdk-js/README.md`](../../sdk-js/README.md): generate a keypair, build
and seal a capsule, write it to disk, verify it from bytes, and read it
back.

```sh
npm install
npm test
```

This example is CI-gated by the conformance harness, so the README's
copy-paste onboarding code is guaranteed to keep running.

No warranty: illustrative only; not a production template.
